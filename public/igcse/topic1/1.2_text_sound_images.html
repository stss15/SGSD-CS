What Is a Character Set?

Character Set

A list of characters matched to numeric codes so text can be stored in binary (e.g., ASCII, Unicode).

Why It Exists
  • ASCII (7-bit) covers basic English keyboard
  • Unicode (UTF-8/16/32) covers languages/emojis globally
Question

Why did we move from ASCII to Unicode?

ASCII is limited to 128 symbols; Unicode supports thousands of scripts/emojis.

How Do We Store Sound?

Digital Audio

Sound captured as regular samples (sample rate) with each sample stored using a fixed number of bits (sample resolution).

Why It Exists
  • Converts continuous waves into data computers can process
  • Enables editing, streaming, and compression
Question

What happens to quality and size if we double the sample rate?

Quality ↑ (captures higher frequencies); file size ↑ proportionally.

How Do We Store Images?

Bitmap Image

A grid of pixels, each pixel encoded with a colour value using a fixed number of bits (colour depth).

Why It Exists
  • Pixels map directly to screen dots
  • Simple math for size and scaling
Question

What changes if we double the colour depth?

More colours, smoother gradients, but file size doubles.

Question 1

"ASCII can store every emoji."

Answer: False

ASCII is 7-bit and limited to basic English symbols; emojis need Unicode encodings like UTF-8.

Question 2

"Doubling the sample rate doubles the sound file size if everything else is constant."

Answer: True

File size is directly proportional to sample rate (and resolution, channels, duration).

Question 3

"Increasing colour depth reduces the image file size."

Answer: False

Higher colour depth uses more bits per pixel, increasing size.
