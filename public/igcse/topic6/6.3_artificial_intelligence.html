Question 1

What are the 3 main characteristics of a robot?

Answer:

Sense (Sensors), Move (Actuators), Programmable (Controller).

Question 2

What is the difference between an independent and a dependent robot?

Answer:

Independent = No direct human control (Autonomous).
Dependent = Human interface required.

Question 3

Give one advantage of using robots in industry.

Answer:

Consistency, Productivity (24/7), Safety (hazardous tasks), or Cost (long term).

Question 4

What is an 'end-effector'?

Answer:

An attachment to a robot arm to perform a specific task (e.g., gripper, spray gun, welder).

Question 5

True or False: A drone controlled by a remote is autonomous.

Answer: False

It is dependent because a human is controlling it.

Sci-Fi vs Reality

The Dream

Sentient machines that think, feel, and take over the world (Skynet, HAL 9000).

The Reality

Algorithms that are really good at spotting patterns in data (Netflix recommendations, Siri, Chess bots).

Key Idea: AI is about simulation, not necessarily consciousness.

What is AI?

What makes a machine "intelligent"?

Artificial Intelligence (AI)

A branch of computer science dealing with the simulation of intelligent human behaviour by a computer.

It mimics cognitive functions: Learning, Reasoning, Problem Solving, Perception.

Deep Dive: Categories of AI

Categories of AI

Are all AIs equal?

Narrow AI
Superior to humans at one specific task.
(e.g., Chess bot, Spam filter).
General AI
Similar performance to a human across many tasks.
(Not yet achieved).
Strong AI
Superior to humans in many tasks.
(The stuff of Sci-Fi).

Expert Systems

How can a computer diagnose illness like a doctor?

Expert System

A computer system that mimics the decision-making ability of a human expert.

Uses
  • Medical Diagnosis
  • Car Engine Fault Finding
  • Mineral Prospecting
  • Chess Strategy
Pros
  • Consistent answers.
  • Never forgets facts.
  • Available 24/7.
Deep Dive: Components

Components of an Expert System

1. Knowledge Base
A database of facts and rules. The "Knowledge" of the expert.
(e.g., IF spots AND fever THEN measles)
2. Rule Base
A set of inference rules (IF-THEN statements) used to draw conclusions.
3. Inference Engine
The "Brain". It applies the rules to the knowledge base and user inputs to find a solution.
4. User Interface
How the user interacts (answers questions) and receives the advice/diagnosis.

Machine Learning (ML)

How does Netflix know what you want to watch?

Machine Learning

A subset of AI where algorithms are trained on data to learn from experience, rather than being explicitly programmed for every rule.

Key Difference:

Traditional Code: You write the rules.
Machine Learning: The computer learns the rules from data.

Deep Dive: AI vs ML

AI vs Machine Learning

Artificial Intelligence

The broad goal of simulating intelligence.

"I want a machine that acts like a human."

Machine Learning

A specific technique to achieve AI using data.

"I will feed this machine 10,000 photos of cats so it learns to recognise cats."

Examples of ML: Spam filters, Voice recognition, Self-driving cars, Recommendation engines.

Question 1

"Machine Learning requires explicit programming for every rule."

Answer: False

It learns from data and experience. Traditional programming requires explicit rules.

Question 2

"An Expert System uses an Inference Engine to make decisions."

Answer: True

The Inference Engine applies the rules from the Rule Base to the facts in the Knowledge Base.

Question 3

"Narrow AI can perform many different tasks better than a human."

Answer: False

Narrow AI is good at one specific task (e.g., playing Chess). Strong AI would be good at many tasks.
