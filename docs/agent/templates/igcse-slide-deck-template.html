Core Concept

System Software

Programs that manage the computer's hardware and provide a platform for application software to run. System software operates in the background and is essential for basic computer functionality.

Write this definition in your notes

Students must write this down verbatim. This is exam vocabulary. TIMING: 2 minutes

Two Main Categories

System Software

Manages hardware and provides platform for applications

  • Operating System (OS)
  • Utility Programs
  • Device Drivers
  • Translators (Compilers)

Application Software

Allows users to perform specific tasks

  • Word Processors
  • Spreadsheets
  • Databases
  • Graphics/Photo Editing
Key distinction: System = background; Application = user-facing. TIMING: 3 minutes

Fix The Bug

This loop doesn't work. What's wrong?

Buggy Code 

count = 0
while count < 5:
    print(count)
# Loop runs forever!
Fixed Code 

count = 0
while count < 5:
    print(count)
    count = count + 1  # Increment!

Always increment your counter in loops!

Common error: forgetting to increment. Ask students to identify the problem first. TIMING: 3 minutes

Compiler vs Interpreter

Compiler

  • Translates whole program at once
  • Creates executable file (.exe)
  • Fast execution
  • Reports errors at end

Interpreter

  • Translates line by line
  • No executable created
  • Slower execution
  • Stops at first error
When to use: Compiler for distribution | Interpreter for development/testing
Crucial distinction for exam. Emphasize: Compiler = finished product; Interpreter = debugging. TIMING: 4 minutes

Real-World Scenario

You've written a video game and want to sell it online. You don't want customers to see or steal your source code. Which translator should you use?

Answer: Compiler

A compiler creates a standalone executable file (.exe). The customer doesn't need the source code or a translator installed. Your code remains proprietary.

Link technical knowledge to real-world application. This is exam-style reasoning. TIMING: 2 minutes

"All system software has a GUI."

Answer: False

Question 2

"An interpreter creates an executable file."

Answer: False

Interpreters run code line-by-line; compilers generate executables.

Question 3

"Interrupts pause the CPU forever."

Answer: False
